# Isha-Salaes-Dashboard

This project presents an interactive Sales Dashboard designed to analyze and monitor business performance across key sales metrics. The dashboard enables stakeholders to track revenue trends, identify top-performing products and regions, and make data-driven decisions.

1. Key Features

Total Sales, Profit & Quantity KPIs for quick performance overview
Time-based analysis (monthly / quarterly / yearly trends)
Top Products & Categories by revenue and profit
Regional & Customer-level insights
Dynamic filters and slicers for interactive exploration

2. Business Insights

Identifies high-revenue and high-margin products
Highlights seasonal sales patterns
Detects underperforming regions or customers
Supports strategic planning and forecasting

3. Tools & Technologies

Power BI (DAX, Power Query, Data Modeling)
SQL / Excel (data preparation)
Star Schema for optimized performance

4. Dataset

Sales transactions including:
Order Date
Product, Category
Customer & Region
Sales, Quantity, Profit

5. Use Case

Ideal for Sales Managers, Business Analysts, and Executives to:
Monitor sales performance
Improve revenue strategies
Optimize product and regional focus
